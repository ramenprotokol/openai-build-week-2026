# OpenAI Build Week 2026

> Ramen Protocol's public entry for OpenAI Build Week 2026.

This repository is the clean build workspace for a working project made with Codex
and GPT-5.6. The product concept is still being selected. The goal is one complete,
useful workflow that judges can understand and test without rebuilding a large
system.

## Challenge constraints

- Submission deadline: July 21, 2026 at 5:00 PM Pacific Time.
- The project must be built with Codex and meaningfully use GPT-5.6.
- The final entry needs a working test path, clear README, public repository, and a
  public demo video no longer than three minutes.
- The demo must explain the product, the Codex workflow, and the GPT-5.6 integration.

## Status

**Planning.** Public identity and repository location are established. Product name,
track, target user, core workflow, and implementation stack are not yet locked.

No product code has been written.

## Development principles

- One target user and one end-to-end workflow for version 0.1.
- GPT-5.6 must perform essential product work rather than appear as decoration.
- Judges must have a simple way to see the product working.
- Every public file and commit must remain safe for the Ramen Protocol identity.
- Scope is reduced before the deadline is moved.

## Built with AI in the loop

Codex will be used throughout planning, implementation, testing, review, and
documentation. The final submission will identify the primary Codex build task and
explain which decisions remained human-owned.
